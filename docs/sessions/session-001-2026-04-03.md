# Session 001 — 2026-04-03

## Cilj seje
Postavitev projekta za Home Assistant custom component integracijo za SPAR Online (online.spar.si).

## Opravljeno

### 1. Raziskava SPAR Online API
- Identificirana platforma: Instaleap "Deadpool" (GraphQL backend)
- Dva endpointa: v2 (katalog) in v3 (e-commerce/cart)
- Avtentikacija: Firebase Auth -> Instaleap JWT, ali direktni v3 signIn
- Dokumentirane vse ključne GraphQL operacije (search, cart CRUD, auth)
- Odkrite konfiguracije: API ključi, client ID, store ID

### 2. Raziskava HA integracije
- Preučena struktura custom components (manifest, config_flow, coordinator, todo)
- TodoListEntity za nakupovalni seznam in košarico
- DataUpdateCoordinator za polling
- Translations v SLO + EN (custom components morajo imeti translations/, ne strings.json)

### 3. Projektna struktura
- Inicializiran git repo (main branch)
- Ustvarjen CLAUDE.md s celotnim kontekstom projekta
- Ustvarjen .gitignore
- Postavljeni direktoriji: custom_components/spar_si/, docs/, tests/

### 4. Memory sistem
- user_profile.md — profil uporabnika
- project_spar_ha.md — opis projekta in ciljev
- feedback_workflow.md — pravila načina dela
- reference_spar_api.md — tehnični API detajli

### 5. Implementacija API klienta (api.py)
- `SparApiClient` z async aiohttp
- Avtentikacija: v3 signIn (email/password -> JWT)
- Iskanje izdelkov: `async_search_products()`
- Košarica: `async_get_or_create_cart()`, `async_add_to_cart()`, `async_update_cart_item()`, `async_remove_from_cart()`
- Avtomatski re-auth ob potečenem tokenu
- Dataclass modeli: `SparProduct`, `SparCartItem`, `SparCart`, `SparCustomer`

### 6. HA Custom Component
- **manifest.json** — domain: spar_si, iot_class: cloud_polling
- **const.py** — vse konstante (API URL-ji, ključi, privzete vrednosti)
- **config_flow.py** — user step (email/password/store) + reauth flow
- **coordinator.py** — DataUpdateCoordinator za cart polling (5 min)
- **__init__.py** — async_setup_entry, async_unload_entry, search_products service
- **todo.py** — 2 entiteti:
  - `SparShoppingListEntity` — lokalni nakupovalni seznam (voice friendly)
  - `SparCartEntity` — sinhronizirana SPAR košarica
- **services.yaml** — search_products service definicija
- **translations/en.json** — angleške prevode
- **translations/sl.json** — slovenske prevode
